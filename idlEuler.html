IdlEuler -- Math Idle Game

If you are not into idle games, this project may seem quite weird to you. The main and only objective is to make a particular number as high as possible. All you can do for that is to buy upgrades which will make it grow faster. But the settings and specific mechanis can make the gamepplay quite interesting and deep. Again, if you are into idle games.

Play the game:

IdlEuler

Here is an overview of main mechanics:

  • In the "Numbers" section you can upgrade the values of variables.
  • In the "Functions" sections you can buy functions using those variables and producing the currency of the game -- Natural numbers.
  • In the "Geometry" section you can buy shapes, which also depend on the variables and their properties serve as production multipliers.
  • The ascention mechanic of the game (common term for idle games, meaning restarting the game with some powerful boosts) is "Writing a book", when you compose all the knowledge you got in a book, reset your progress, but get some axioms for that.
  • All the books you've written can be found in the "Library".

The next feature I'm working on is "Theorems" -- powerful upgrades you buy for axioms which boost your functions and shapes.
